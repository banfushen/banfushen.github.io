Ubuntu 18.04.3 安装redis，配置端口与密码

发表于 2019-08-12 更新于 2023-07-16 分类于 redis

实际开发中，经常需要配置redis的端口与密码，硬盘坏了，换个硬盘，重新装开发环境

1.sudo apt-get install redis-server

2.查看是否安装成功，安装成功，会自动启动

ps -aux | grep redis

3.修改端口

sudo subl /etc/redis/redis.conf

将6379改为自己需要的端口。然后保存

4.重启redis，然后查看

sudo service redis-server restart

ps -aux | grep redis

已修改成功。

5.继续修改redis.conf

找到一个 # requirepass foobared 的字段将这个字段的注释取消掉，这个字段是数据的访问密码，将foobared替换成自己想要设置的密码。

修改之后

6.重启redis

sudo service redis-server restart
