注意：

1.table.sort并不稳定，当条件的两个元素相等时，它们在排序后的相对位置可能会改变(据说，我自己暂未出现，可能用的少)

2.要求需要排序table中间元素不能有nil，否则会报错

3.当比较的两个元素相等的时候，比较函数一定要返回false，返回true会报错，table.sort会根据你返回的bool来判断两个value是否保持原来的顺序
